## 中文

编辑 `user.csv` 配置用户名

编辑 `config.ini` 可以配置水印密度和后缀

PDF 文件放到 `input/` 文件夹下

运行 `main.exe` 会给文件加水印并输出到 `out/` 目录下，原始文件会被移动到 `done/` 目录下

---

## English

Edit `user.csv` to configure usernames

Edit `config.ini` to configure watermark density and suffix

Put PDF files in the `input/` folder

Run `main.exe` to add watermarks and output to the `out/` directory, original files will be moved to the `done/` directory
